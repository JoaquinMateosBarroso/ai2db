Estudio de objetivo y viabilidad para herramienta de automatización de consultas a Bases de Datos¶

import pandas as pd
import matplotlib.pyplot as plt

from pytrends.request import TrendReq
pytrend = TrendReq()

Solicitamos a Google datos sobre los tipos de bases de datos más usadas a día de hoy.¶

# Ask google for data about diverse databases
pytrend.build_payload(kw_list=["SQL", "noSQL", "MongoDB", "Casandra", "CouchDB"])

# Interest over time
df_overtime = pytrend.interest_over_time()

# Plot
df_overtime.plot()
plt.legend(loc='best')
plt.show()

Vemos que SQL siempre ha dominado las b√∫squedas, y eso se corrobora en estudios sobre su empleo en empresas.

Por ende, si con nuestra herramienta perseguimos alcanzar y ayudar al mayor número posible de personas, lo más inteligente será optimizar SQL.¶

pytrend.build_payload(kw_list=["AI SQL", "easy SQL", "manage database"])

# Interest by Region
df_reg = pytrend.interest_by_region()
df_reg.to_csv('interest_by_region2.csv')

# Interest over time
df_overtime = pytrend.interest_over_time()
df_overtime.to_csv('interest_over_time2.csv')

# Make a pie about interest over time
plt.pie(df_overtime.mean()[0:-1], labels = df_overtime.axes[1][0:-1], autopct='%1.1f%%')
plt.title('Porcentaje de b√∫squedas a la hora de consultar sobre SQL en Google')
plt.axis('equal')
plt.show()

plt.plot(df_overtime.index, df_overtime['AI SQL'], label='AI SQL')
plt.plot(df_overtime.index, df_overtime['easy SQL'], label='easy SQL')
plt.plot(df_overtime.index, df_overtime['manage database'], label='manage database')
plt.legend(loc="upper left")

plt.title('Interest over time')
plt.show()
